## 选股标准（可量化）

### 必须条件（硬筛选）
```python
def hard_filter(stock):
    """硬筛选条件（必须全部满足）"""
    checks = [
        ("实控人", stock.controller in ["央企", "省国资委", "市国资委"]),
        ("市值", stock.market_cap < 200_0000_0000),  # <200亿
        ("主营集中", stock.main_business_pct > 70),
        ("PB", stock.pb < 2.5),
        ("非ST", not stock.is_st),
        ("非民企", stock.controller not in ["民企", "无实控人"]),
    ]
    return all(check[1] for check in checks)
```

### 加分条件（软筛选）
```python
def soft_filter(stock):
    """软筛选条件（加分项）"""
    scores = {
        "PB破净": 20 if stock.pb < 1.0 else 0,
        "低位盘整": 15 if stock.consolidation_years >= 2 else 0,
        "高股息": 15 if stock.dividend_yield > 3 else 0,
        "业绩爆发": 20 if stock.profit_growth > 300 else 0,
        "稀缺性": 15 if stock.is_unique else 0,
        "央企背景": 10 if stock.controller == "央企" else 5,
    }
    return sum(scores.values())
```
