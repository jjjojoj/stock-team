## 每日循环

```
09:00  早盘预测（使用最新规则权重）
       ↓
10:30  盘中监控（检测止损/止盈触发）
       ↓
13:00  午盘更新（根据上午走势调整预测）
       ↓
15:00  收盘前检查（最后交易机会）
       ↓
15:30  盘后复盘
       ├── 验证所有预测
       ├── 计算准确率
       ├── 更新规则权重
       ├── 验证池规则晋升/废弃
       └── 记录教训
       ↓
20:00  深度学习（读书 + 提取规则）
       ↓
次日 09:00  循环开始（使用更新后的规则）
```

## 每周循环

```
周日 20:00  周总结
       ├── 本周准确率统计
       ├── 最佳/最差规则
       ├── 选股标准回顾
       ├── 止盈止损效果评估
       └── 下周策略调整
```

## 文件结构

```
china-stock-team/
├── config/
│   ├── strategy.md          # 投资策略（手动维护）
│   ├── portfolio.json       # 资金状态
│   └── schedule.json        # 定时任务配置
├── learning/
│   ├── memory.md            # 长期记忆（教训/经验）
│   ├── rule_validation_pool.json  # 验证池（待验证规则）
│   ├── prediction_rules.json      # 规则库（实际运用）
│   ├── accuracy_stats.json        # 准确率统计（闭环核心）
│   └── rule_evolution_history.json # 规则演变历史
├── data/
│   ├── predictions.json     # 预测记录
│   └── reviews/             # 复盘报告
└── scripts/
    ├── daily_prediction_loop.py    # 每日预测循环（早/午/晚）
    ├── daily_review_closed_loop.py # 复盘+闭环反馈
    ├── rule_promotion.py           # 规则晋升/废弃
    └── scheduler.py                # 24小时调度器
```
