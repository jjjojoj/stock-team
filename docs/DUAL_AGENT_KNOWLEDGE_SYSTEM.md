## 🤖 Agent 1: 搜集官（Collector）

### 职责

**任务**：爬取网络资源，存入知识库

### 数据源

**1. 书籍资源**
- 豆瓣读书 - 高分投资书籍
- 京东图书 - 新书推荐
- 微信读书 - 热门标注

**2. 论坛社区**
- 雪球 - 大V观点
- 东方财富 - 热门帖子
- 知乎 - 投资话题
- 贴吧 - 散户经验

**3. 财经媒体**
- 华尔街见闻 - 深度分析
- 财新网 - 宏观研究
- 新浪财经 - 市场评论

**4. 学术资源**
- SSRN - 学术论文
- Arxiv - 量化研究
- Google Scholar - 引用追踪

### 工作流程

```
1. 每日定时爬取（晚上8点）
   ↓
2. 去重过滤（避免重复）
   ↓
3. 质量评分（筛选高质量）
   ↓
4. 存入知识库（标记new）
   ↓
5. 发送通知（今日收集X条）
```

### 配置示例

```json
{
  "sources": [
    {
      "name": "雪球热帖",
      "url": "https://xueqiu.com/hot",
      "type": "post",
      "frequency": "daily",
      "filters": {
        "min_likes": 100,
        "min_comments": 50
      }
    },
    {
      "name": "豆瓣读书-投资",
      "url": "https://book.douban.com/tag/投资",
      "type": "book",
      "frequency": "weekly",
      "filters": {
        "min_rating": 8.0
      }
    }
  ]
}
```

## 🎓 Agent 2: 学习官（Learner）

### 职责

**任务**：阅读知识库，浓缩总结成规则

### 工作流程

```
1. 扫描知识库（查找status=new）
   ↓
2. AI阅读理解（提取核心观点）
   ↓
3. 转化可测试规则（设计验证形式）
   ↓
4. 去重检查（避免重复规则）
   ↓
5. 存入验证池（等待市场验证）
   ↓
6. 更新知识库状态（processed）
```

### 转化示例

**输入**（知识库原文）：
```
"趋势交易的核心是顺势而为，不要试图抄底摸顶。
当股价突破20日均线且成交量放大时，往往意味着趋势确立。"
```

**输出**（验证池规则）：
```json
{
  "rule_id": "rule_trend_001",
  "rule": "突破20日均线+成交量放大 → 趋势确立",
  "testable_form": "突破MA20且volume>2倍均值，5日内上涨概率>60%",
  "category": "趋势",
  "source": "雪球热帖-趋势交易的核心逻辑",
  "live_test": {
    "samples": 0,
    "success_rate": 0
  },
  "status": "validating"
}
```

### 智能特性

**1. 多角度转化**
- 一篇文章 → 3-5条候选规则
- 不同时间周期（短期/中期/长期）
- 不同市场环境（牛市/熊市/震荡）

**2. 规则合并**
- 相似规则自动合并
- 冲突规则标记对比
- 优先级排序

**3. 失败重设计**
- 规则验证失败后
- 回到知识库重新设计
- 生成新的可测试形式

## 🔄 协作流程

### 每日流程

```
20:00 - 搜集官启动
  ↓ 爬取网络资源
  ↓ 去重过滤
  ↓ 存入知识库（50条新知识）
  
21:00 - 学习官启动
  ↓ 扫描知识库
  ↓ AI阅读理解
  ↓ 转化可测试规则
  ↓ 存入验证池（15条新规则）
  
次日09:00 - 预测系统
  ↓ 使用验证池规则
  ↓ 生成预测
  ↓ 记录规则使用
  
15:30 - 复盘系统
  ↓ 验证预测
  ↓ 更新规则统计
  
16:00 - 规则晋升
  ↓ 检查验证池
  ↓ 样本≥15+胜率≥60%
  ↓ 晋升到规则库
```

## 📊 知识流转

```
【第一层：收集】
搜集官 → 50条/天 → 知识库

【第二层：转化】
学习官 → 15条/天 → 验证池

【第三层：验证】
预测系统 → 使用规则 → 15样本

【第四层：晋升】
验证池 → 60%胜率 → 规则库
```

## 🔧 技术实现

### 搜集官技术栈

```python
- requests + BeautifulSoup（爬虫）
- newspaper3k（文章提取）
- readability-lxml（正文提取）
- 去重：simhash
- 存储：JSON
```

### 学习官技术栈

```python
- OpenAI API（阅读理解）
- 规则模板（可测试形式）
- 去重：规则相似度计算
- 存储：JSON
```
