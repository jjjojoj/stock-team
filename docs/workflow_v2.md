## 📅 每日工作流程

### 09:00 开盘前准备

**任务**: `开盘前联网搜索` (cron: 0 9 * * 1-5)

```bash
python3 scripts/daily_web_search.py
```

**执行内容**:
1. 搜索最新新闻（宏观 + 行业 + 个股）
2. 分析新闻对持仓股的影响
3. 生成当日预测（持仓 + 观察池，最多 27 只）
4. 飞书汇报：重要新闻 + 预测汇总

**输出**:
- 新闻摘要
- 持仓股影响分析
- 当日预测表

---

### 09:30 早盘开盘

**任务**: 
- `早上 AI 预测生成` (09:30)
- `每日预测扫描` (09:30)
- `持仓汇报 - 早盘开盘` (09:35)

```bash
python3 scripts/ai_predictor.py brief
python3 scripts/prediction_engine.py scan
python3 scripts/price_report.py
```

**执行内容**:
1. 为持仓和观察池生成 AI 预测
2. 扫描高置信度股票（≥80% 自动买入）
3. 发送持仓汇报到飞书

**自动买入规则**:
- 置信度 ≥ 80% → 自动买入
- 置信度 60-79% → 加入观察
- 置信度 < 60% → 观望

---

### 09:30-11:30 早盘交易

**监控任务**:
- `涨跌停监控` (09:25, 14:25) - 每日 2 次
- `熔断紧急减仓` (09:25, 14:25) - 每日 2 次
- `新闻监控 - 预测更新` (10:00, 12:00, 14:00)

**自动执行**:
- 涨跌停预警 → 飞书通知
- 熔断触发 → 紧急减仓
- 重大新闻 → 更新预测

---

### 11:30 午盘反思 ⭐ **核心学习环节**

**任务**: `午盘反思` (cron: 30 11 * * 1-5)

```bash
python3 scripts/midday_review.py
```

**执行内容**:
1. 验证早盘预测准确率
2. 总结：哪里对/哪里错
3. **永久写入学习记忆**（影响所有未来预测）
   - 写入 `learning/memory.md`（HOT 层）
   - 调整 `prediction_config.json`（全局参数）
   - 记录到日志（可追溯）
4. 飞书汇报：准确率 + 教训

**关键**: 教训不是只应用到下午，而是**永久生效**，从明天开始的所有预测都会受影响

**输出**:
- 预测准确率统计
- 错误案例分析
- **学习记忆更新**（永久）
- 配置参数调整（永久）

---

**自动执行**:
- 自动买卖（基于更新后的预测）
- 风险监控

---

### 15:00 收盘

**任务**:
- `股票每日扫描` (15:00)
- `持仓汇报 - 收盘汇总` (15:05)
- `每日调仓检查` (15:10)

---

### 15:30 收盘后深度复盘 ⭐ **最核心学习环节**

**任务**: `收盘后深度复盘` (cron: 30 15 * * 1-5)

```bash
python3 scripts/daily_review_system.py report
python3 scripts/daily_learning.py
python3 scripts/stock_pool_manager.py auto
```

**执行内容**:
1. **验证全天预测准确率**
   - 正确/错误/部分正确
   - 分股票统计
   - 分行业统计

2. **深度学习：成功/失败案例分析**
   - 成功案例：为什么对？→ 提取成功模式
   - 失败案例：为什么错？→ 提取失败教训
   - 提取规律 → 永久写入记忆

3. **优化选股逻辑（永久生效）**
   - 更新 `prediction_engine.py` 规则权重
   - 调整置信度阈值
   - **写入 `learning/memory.md`（HOT 层 - 永久生效）**
   - 所有未来预测都会应用这些教训

4. **股票池管理**
   - 评分排序所有股票
   - 淘汰最差股票（评分<40 或不符合逻辑）
   - 补充新股票（从 stock_pool.md）
   - 保持池子大小：15-20 只

5. **更新投资框架文档**
   - 记录今日教训
   - 更新选股标准
   - 保存到 `config/strategy.md`

6. **飞书汇报**
   - 准确率统计
   - 学到的教训（永久生效）
   - 股票池调整

**关键**: 这是**最重要的学习环节**，今天的教训会变成明天的智慧，永久影响所有未来预测

## 📊 股票池管理系统

### 容量规则
- **上限**: 20 只股票
- **下限**: 15 只股票（低于则补充）
- **淘汰机制**: 评分最低 + 不符合逻辑

### 评分标准
```python
score = (
    预测置信度 * 40% +      # 高置信度加分
    近期预测准确率 * 30% +   # 历史表现
    持仓表现 * 20% +        # 盈亏情况
    行业景气度 * 10%        # 行业周期
)
```

### 淘汰流程
1. 计算所有股票评分
2. 找出最低分（<40 分）
3. 移动到 `eliminated_pool.json`
4. 从候选池补充新股

### 候选来源
- `config/stock_pool.md` 未加入的股票
- 每日深度研究新发现的股票

## 📚 学习系统（核心竞争优势）

### 学习触发点
1. **预测错误** → 记录失败原因 → 永久写入记忆
2. **预测正确** → 记录成功因素 → 永久写入记忆
3. **市场变化** → 调整规则权重 → 永久生效
4. **新股加入** → 更新选股标准 → 永久生效

### 学习记忆（HOT 层 - 永久生效）
- **位置**: `learning/memory.md`
- **特点**: 
  - ⭐ **永久生效** - 不是临时调整
  - ⭐ **所有未来预测都会应用** - 从明天开始
  - ⭐ **可追溯** - 记录学习时间和原因

- **格式**:
  ```markdown
  ## [2026-03-09 11:30] 午盘学习 ⭐
  
  ⚠️ **ERROR**: 早盘上涨预测准确率仅 30%
     - **行动**: 提高置信度阈值
     - **影响范围**: 所有未来预测（永久生效）
     - **应用时机**: 午盘反思 → 立即写入记忆 → 明日起生效
  
  ## [2026-03-09] 收盘深度学习 ⭐
  
  ✅ **SUCCESS**: 北方稀土预测正确 (+5%)
     - 成功因素：行业周期低位 + 央企背景
     - 影响：所有未来预测（永久生效）
  ```

### 规则进化
- **位置**: `prediction_engine.py` 的 `PREDICTION_RULES`
- **机制**: 
  - 每条规则记录成功率
  - 成功率<40% → 自动降低权重
  - 成功率<30% → 标记为待移除
- **更新**: 
  - 每日：午盘反思 + 收盘复盘
  - 每周：回测验证 + 规则回顾

### 学习闭环

```
预测 → 验证 → 学习 → 永久记忆 → 改进预测 → 验证 → ...
  ↑                                              ↓
  └──────────────────────────────────────────────┘
  
每一次循环，系统都变得更聪明
```

## 🎯 关键指标

### 预测准确率
- **目标**: > 60%
- **计算**: 正确预测数 / 总预测数
- **跟踪**: 每日记录，周/月统计

### 股票池健康度
- **平均评分**: > 50 分
- **淘汰率**: 每周 1-2 只
- **更新率**: 每周 5-10%

### 投资回报
- **持仓盈亏**: 每日汇报
- **超额收益**: 相对基准指数
- **最大回撤**: < 15%

## 🚨 异常处理

### API 故障
- **自动切换**: 5 个新闻 API + 4 个行情 API
- **健康检查**: 每日 3 次 (09:00, 11:00, 15:00)

### 预测连续失败
- **阈值**: 连续 3 天准确率 < 40%
- **应对**: 降低仓位，暂停自动买入
- **复盘**: 深度分析框架问题

### 市场极端情况
- **熔断**: 自动减仓 50%
- **暴跌**: 止损机制触发
- **黑天鹅**: 人工介入
