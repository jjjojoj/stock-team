## [2026-03-03] 调仓规则（团队自主决策）

**决策权**：团队完全自主，用户只知情不参与

**调仓触发**：每日收盘后自动检查

**调仓规则**：
```python
if pnl_pct < -3 and today_change < 0:
    action = "止损卖出"
    reason = f"亏损{pnl_pct}%且今日下跌{today_change}%"
elif pnl_pct < -3 and today_change > 0:
    action = "持有观望"
    reason = f"亏损但今日上涨，等反弹"
else:
    action = "持有"
    reason = "正常持有"
```

**仓位控制**：
- 单只 ≤ 20%资金
- 单行业 ≤ 40%资金
- 总仓位 ≤ 80%

**知情通知**：每次调仓后飞书通知用户

## [2026-03-03] 看多预测集体失败

**预测结果**：4只看多 → 4只全错（收盘均下跌）

**失败原因**：

1. **预测时机错误**
   - 预测在 12:17 生成（盘中高点）
   - 当时价格已大涨，收盘回调
   - **教训**：预测应在开盘前(09:25)或收盘后(15:05)

2. **目标价太激进**
   - 统一 +20% 目标价
   - 1周周期内 +20% 不现实
   - **教训**：目标价应根据置信度调整（高70%→+10%，中50%→+5%，低→+3%）

3. **忽视技术面超买**
   - 贵研铂业 RSI 76，中色股份 RSI 74.3
   - 没有纳入置信度计算
   - **教训**：RSI > 70 应降低置信度10%，标记超买风险

4. **忽视当日涨幅**
   - 预测时当日已大涨 25%~61%
   - 没有考虑追高风险
   - **教训**：当日涨 >5% 应降低看多置信度15%

**成功案例**：
- 中色股份 → 观望 ✅ 正确
- 原因：置信度只有58%，不是强烈信号

**可提取规则**：
1. 预测生成时间改为 09:25（开盘前）或 15:05（收盘后）
2. 目标价按置信度分层：高+10%，中+5%，低+3%
3. RSI > 70 降低置信度10%，标记超买风险
4. 当日涨 >5% 降低看多置信度15%，标记追高风险
